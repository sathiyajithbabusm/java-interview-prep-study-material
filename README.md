# Java Interview Prep Study Material

☕ A comprehensive, single-page study guide for developers targeting top tech companies (Amazon, Walmart, Google, JPMorgan, Flipkart, and similar).

**Live site:** https://sathiyajithbabusm.github.io/java-interview-prep-study-material/

## What's inside

72 in-depth Q&A entries across 8 categories:

- 🔵 Core Java & OOP
- 📦 Collections Framework
- 🔴 Multithreading & Concurrency
- ⚡ JVM Internals & Performance
- ☕ Java 8–21 Features
- 🎨 Design Patterns
- 🌿 Spring & Microservices
- 🗄 Databases, Caching & Architecture

Each question includes a detailed answer and, where relevant, a runnable code snippet.

## Features

- 🔍 Live search across questions, answers, and categories
- 📋 Quick navigation sidebar
- ⌨️ Keyboard-accessible cards and navigation
- 🖨 Print / Save as PDF
- No build step, no dependencies — a single self-contained `index.html`

## Running locally

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000/index.html`.

## Deployment

Pushes to `main` automatically deploy to GitHub Pages via [`.github/workflows/static.yml`](.github/workflows/static.yml).
